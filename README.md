# Lynq Backoffice (Next.js)

Production-ready Next.js + Tailwind + TypeScript implementation of the Figma backoffice dashboard design.

## Stack

- Next.js App Router
- TypeScript (strict)
- Tailwind CSS
- shadcn-style UI primitives (Radix + CVA)
- Recharts for dashboards
- React DnD for candidate board interactions

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` Overview
- `/job-cards`
- `/job-cards/[id]/edit`
- `/job-cards/[id]/ab-test`
- `/sponsored`
- `/candidates`
- `/candidates/review`
- `/candidates/[id]`
- `/analytics`
- `/market-insights`
- `/billing`
- `/settings`
